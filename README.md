# Campfire
A decentralized and customizable chatting client.

### Testing
(Requires Node & npm)

 1. Clone this repo and `cd` to it.
 2. Install dependencies `npm install`
 3. For non-simple: `npm run campfire`<br>
    For simple: `npm run simple`
